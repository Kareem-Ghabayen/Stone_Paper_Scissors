# 🪨 Stone Paper Scissors Project (C++)

This is a simple implementation of the **Stone Paper Scissors** game built with **C++**.

## 🚀 About the Project
- **Language:** C++.
- **Purpose:** Applying programming concepts learned from the Elzero course.
- **Efficiency:** Only essential source files (.cpp) and solution files (.slnx) are included to keep the repository lightweight.

## 📂 Repository Structure
- `Stone/Stone/Stone.cpp`: The main source code containing the game logic.
- `Stone/Stone.slnx`: The solution file to open the project in Visual Studio.

## 🛠️ How to Run
1. Download the repository as a ZIP or use `git clone`.
2. Open the `Stone.slnx` file using **Visual Studio**.
3. Build and run the project.
